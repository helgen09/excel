Задача для джуниора:
Реализовать импорт данных из Excel-файла в базу данных с валидацией. Данные должны быть сохранены в таблицу rows. Ошибки валидации записываем в таблицу import_logs, в конце импорта записываем логи импорта в файл result.txt. Прогресс импорта должен сохраняться в Redis. После импорта данные должны быть доступны через RESTful API с группировкой по дате согласно утвержденному формату. Также необходимо настроить передачу событий через Laravel Echo.
